00032     //                 x
00033     //              2      |
00034     //     erf(x)  =  ---------  | exp(-t*t)dt
00035     //           sqrt(pi) \|
00036     //                 0
00037     //
00038     //     erfc(x) =  1-erf(x)
00039     //  Note that
00040     //      erf(-x) = -erf(x)
00041     //      erfc(-x) = 2 - erfc(x)
00042     //
00043     // Method:
00044     //  1. For |x| in [0, 0.84375]
00045     //      erf(x)  = x + x*R(x^2)
00046     //          erfc(x) = 1 - erf(x)           if x in [-.84375,0.25]
00047     //                  = 0.5 + ((0.5-x)-x*R)  if x in [0.25,0.84375]
00048     //     where R = P/Q where P is an odd poly of degree 8 and
00049     //     Q is an odd poly of degree 10.
00050     //                       -57.90
00051     //          | R - (erf(x)-x)/x | <= 2
00052     //
00053     //
00054     //     Remark. The formula is derived by noting
00055     //          erf(x) = (2/sqrt(pi))*(x - x^3/3 + x^5/10 - x^7/42 + ....)
00056     //     and that
00057     //          2/sqrt(pi) = 1.128379167095512573896158903121545171688
00058     //     is close to one. The interval is chosen because the fix
00059     //     point of erf(x) is near 0.6174 (i.e., erf(x)=x when x is
00060     //     near 0.6174), and by some experiment, 0.84375 is chosen to
00061     //     guarantee the error is less than one ulp for erf.
00062     //
00063     //      2. For |x| in [0.84375,1.25], let s = |x| - 1, and
00064     //         c = 0.84506291151 rounded to single (24 bits)
00065     //  erf(x)  = sign(x) * (c  + P1(s)/Q1(s))
00066     //  erfc(x) = (1-c)  - P1(s)/Q1(s) if x > 0
00067     //            1+(c+P1(s)/Q1(s))    if x < 0
00068     //  |P1/Q1 - (erf(|x|)-c)| <= 2**-59.06
00069     //     Remark: here we use the taylor series expansion at x=1.
00070     //      erf(1+s) = erf(1) + s*Poly(s)
00071     //           = 0.845.. + P1(s)/Q1(s)
00072     //     That is, we use rational approximation to approximate
00073     //          erf(1+s) - (c = (single)0.84506291151)
00074     //     Note that |P1/Q1|< 0.078 for x in [0.84375,1.25]
00075     //     where
00076     //      P1(s) = degree 6 poly in s
00077     //      Q1(s) = degree 6 poly in s
00078     //
00079     //      3. For x in [1.25,1/0.35(~2.857143)],
00080     //  erfc(x) = (1/x)*exp(-x*x-0.5625+R1/S1)
00081     //  erf(x)  = 1 - erfc(x)
00082     //     where
00083     //      R1(z) = degree 7 poly in z, (z=1/x^2)
00084     //      S1(z) = degree 8 poly in z
00085     //
00086     //      4. For x in [1/0.35,28]
00087     //  erfc(x) = (1/x)*exp(-x*x-0.5625+R2/S2) if x > 0
00088     //          = 2.0 - (1/x)*exp(-x*x-0.5625+R2/S2) if -6<x<0
00089     //          = 2.0 - tiny        (if x <= -6)
00090     //  erf(x)  = sign(x)*(1.0 - erfc(x)) if x < 6, else
00091     //  erf(x)  = sign(x)*(1.0 - tiny)
00092     //     where
00093     //      R2(z) = degree 6 poly in z, (z=1/x^2)
00094     //      S2(z) = degree 7 poly in z
00095     //
00096     //      Note1:
00097     //     To compute exp(-x*x-0.5625+R/S), let s be a single
00098     //     precision number and s := x; then
00099     //      -x*x = -s*s + (s-x)*(s+x)
00100     //          exp(-x*x-0.5626+R/S) =
00101     //          exp(-s*s-0.5625)*exp((s-x)*(s+x)+R/S);
00102     //      Note2:
00103     //     Here 4 and 5 make use of the asymptotic series
00104     //            exp(-x*x)
00105     //      erfc(x) ~ ---------- * ( 1 + Poly(1/x^2) )
00106     //            x*sqrt(pi)
00107     //     We use rational approximation to approximate
00108     //  g(s)=f(1/x^2) = log(erfc(x)*x) - x*x + 0.5625
00109     //     Here is the error bound for R1/S1 and R2/S2
00110     //  |R1/S1 - f(x)|  < 2**(-62.57)
00111     //  |R2/S2 - f(x)|  < 2**(-61.52)
00112     //
00113     //      5. For inf > x >= 28
00114     //  erf(x)  = sign(x) *(1 - tiny)  (raise inexact)
00115     //  erfc(x) = tiny*tiny (raise underflow) if x > 0
00116     //          = 2 - tiny if x<0
00117     //
00118     //      7. Special case:
00119     //  erf(0)  = 0, erf(inf)  = 1, erf(-inf) = -1,
00120     //  erfc(0) = 1, erfc(inf) = 0, erfc(-inf) = 2,
00121     //      erfc/erf(NaN) is NaN

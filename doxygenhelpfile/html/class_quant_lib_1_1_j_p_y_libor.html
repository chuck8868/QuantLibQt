Detailed Description

JPY LIBOR rate

Japanese Yen LIBOR fixed by BBA.

See <http://www.bba.org.uk/bba/jsp/polopoly.jsp?d=225&a=1414>.

Warning:
This is the rate fixed in London by BBA. Use TIBOR if you're interested in the Tokio fixing.
